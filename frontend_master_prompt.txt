1. Color Theme - ONLY Use These Classes
Use ONLY these predefined theme colors via Tailwind classes. No hex codes.
primary: #145B47
primaryLight: #1E8E54
secondary: #ECF7F0
secondaryDark: #316F5D
background: #F6FBF6
success: #1E8E54
warning: #F2C94C
error: #E75A4C
2. Border Radius - Strict Theme Usage
rounded-small: 4px
rounded-medium: 6px
rounded-large: 8px
rounded-veryLarge: 12px
Exception: rounded-full is permitted ONLY for perfect circles (avatars, icon backgrounds, dots).
3. Scalable Focus & Interactions
Every interactive element (inputs, buttons) must have a scalable focus state to ensure professional feedback:
Pattern: focus:outline-none focus:ring-2 focus:ring-primary/20 focus:scale-[1.02] transition-all duration-200
Hover: hover:scale-[1.02] active:scale-[0.98]
4. Text Sizing
Use standard Tailwind utilities: text-xs, text-sm, text-base, text-lg, text-xl, etc.
NEVER hardcode pixels like text-[15px].
